# Responsive Sidebar Menu - CodingLab

Uma barra lateral de navegação (Sidebar Menu) moderna, responsiva e elegante desenvolvida com HTML5 e CSS3. O projeto apresenta um menu translúcido sobreposto a um fundo escuro, ideal para painéis administrativos (dashboards) e sistemas web que demandam uma navegação limpa e intuitiva.

##  Tecnologias Utilizadas

* **HTML5:** Estruturação semântica dos links e itens de navegação.
* **CSS3:** Estilização avançada, transições suaves e posicionamento fixo.
* **Efeito Glassmorphism:** Uso de transparências e desfoque de fundo (`backdrop-filter: blur`).
* **Ícones integrados:** Utilização de uma biblioteca de ícones para identificação visual de cada seção.

## Funcionalidades e Técnicas Aplicadas

* **Navegação Completa:** Menu estruturado com seções essenciais, incluindo Dashboard, Shortcuts, Overview, Events, About, Services, Contact e Feedback.
* **Design Translúcido:** Aplicação do efeito de vidro fosco na barra lateral para garantir sofisticação e legibilidade sobre imagens ou fundos complexos.
* **Efeitos de Interação:** Transições animadas suaves aplicadas aos itens do menu durante o comportamento de *hover*.
* **Organização Limpa:** Alinhamento preciso utilizando Flexbox para garantir espaçamento uniforme entre os ícones e os textos.
